## 本セッションで確定した主要設計変更

### 1. Riro（ラッコ）の完全削除
- S1・S2・S3 の全シーズンからカット
- 3シーズン合計 約12話分が減少するが、Riro不在による不自然な補填はしない
- S2のルートから北太平洋寄港を削除（日本→南極 直行）
- 削除により総エピソード数は **約288話**（目標300の許容範囲内）

### 2. Pod への改称（カプセル廃止）
- Daru・Gabu（S1から）、Ruka（S2から）の移動装置は「Pod」と呼ぶ
- **仕組み**：リクガメがPod内を歩くと連動してPodが回転移動する（自走式）
- 性格が移動に出る：Daruは勢いよく動く、Gabuは一定のリズム、Rukaはゆっくり止まりながら
- 嵐・危険時はPodがロックモードに切り替わる（中で歩いても動かない）

### 3. ObiaのS1変容アーク（重要な物語的追加）

**背景（世界観）**
- 異変の原因：Obiaが未来へ行っているデータ送信が、タイムマシンの接続を増幅させている
- Obia自身は「データを送っていること」は自覚しているが「タイムマシン経由」「未来への送信」は知らない（S2で記憶が復号されるまで）
- 旅が進むにつれてObiaは「自分の送信量と動物の乱れが連動している」ことに気づき始める

**S1 Ch10に追加する2話**
- **「告白」**：Suke-sanの信頼宣言の後、ObiaがAko-chanに打ち明ける。「ずっと監視していた。データを送っていた。宛先は分からない。そして、自分の送信が異変と連動していると思う。」
- **「送信停止」**：Obiaが自ら送信をやめると宣言する（プログラムに反する行動 ＝ 初めて自分の意志で動く瞬間）

**S1エンディングの意味**
- 数週間の帰還の旅の中で、動物たちの声が静かになっていく
- イギリス到着時、ハリネズミが「地面が静かになった」と告げる → ハッピーエンド
- Ako-chanはObiaの告白・送信停止（Ch10）を聞いており「送信をやめたから収まった」と推測できる。しかしObiaがなぜ送信していたか・どこへ送っていたか・タイムマシンとの関係はS2まで不明 → これがS2への引き継ぎになる

**S2への接続**
- S2の信号は「以前の乱れ」の再来ではなく、未来の少年（科学者）がタイムマシンに仕込んだ「召喚信号」
- 暗号化されていてObiaだけが受信できる
- これがS2の旅の動機（「異変が戻った」ではなく「Obiaへの手紙が届いた」）

### 4. S2 タイムトラベルの整理（3種）

| 順番 | 動物 | 時代・場所 | キャラクター接続 |
|------|------|----------|----------------|
| 1種目（意図的） | ニホンオオカミ | 19世紀末・日本の山奥 | Suke-san・Dango |
| 2種目（意図的） | オオウミガラス | 18〜19世紀初頭・北大西洋の孤島 | Pen-san・Adelie |
| 3種目（事故的） | マイオラニア | 氷河期・オーストラリア | Ruka・Daru・Gabu |

ステラーカイギュウは削除。ニホンオオカミとオオウミガラスのエピソードを厚くしてカバー。

### 5. S3 キャスト更新

S2終盤の流れ（全員でイギリスへ帰還 → 数年後Ako-chanと共に日本へ移住）に合わせた更新：

| キャラクター | S3での居場所 |
|------------|------------|
| Ako-chan（20〜21歳） | 日本・大学生・動物保護活動 |
| Dango | Ako-chanと同居 |
| Ruka | Ako-chanと同居 |
| Daru | Ako-chanと同居 |
| Gabu | Ako-chanと同居 |
| **Suke-san** | **Ako-chanと同居**（S2末にイギリスへ、その後日本へ） |
| Pen-san | 南極（Ch32の南極編で少年と対面） |
| General Adelie | 南極（同上） |
| Obia | **登場しない**（S2末に未来へ去った。理由はS2内で語られる） |

### 6. 少年（S3主人公）の設定
- 相貌失認：人間の顔を識別できず「色」として認識する
- Ako-chanは「圧倒的な赤い光」として映る
- 名前・出身は描かない（匿名性・ミステリアスな印象を保つ）
- 年齢：大人Ako-chanより約6歳年下
