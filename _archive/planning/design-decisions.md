## 6. 異変の正体（物語の核心）

**南極の氷の中に埋まったタイムマシン**が過去と未来の間の接続を保ったままになっている。未来の世界では種の絶滅と環境の劣化が進行しており、その「崩壊に向かいつつある未来の状態」がタイムマシンの接続を通じて過去へと漏れ出している。これが異変の正体。南極から北へ波及し、南に行くほど影響が強い。

**動物たちへの影響**：風・潮の流れ・匂いの変化、渡り鳥が方向を見失う、クジラが浅瀬に迷い込む、季節のずれ、として現れる。

**Obiaの立場**：S1では自分がなぜここにいるか分からない（記憶が暗号化されている）。「南が原因」とだけ推測して旅を勧める。S2でタイムマシンを発見し記憶が復号、使命を知る：「絶滅しゆく動物たちの声を救え。そのデータを未来に転送せよ。」

## 7. ObiaのS1変容アーク

### 背景（異変の二重構造）

| 要素 | 内容 |
|------|------|
| 常時ノイズ | タイムマシンの接続自体が弱い背景ノイズを生んでいる |
| 増幅要因 | **Obiaのデータ送信**がタイムマシン接続を通じて異変を増幅させている |
| Obiaの自覚 | 「データを送っている」ことは知っている。「タイムマシン経由」「未来への送信」はS2まで知らない |
| Obiaの気づき | 旅が進むにつれ「自分の送信量と動物の乱れが連動している」ことを確信していく |

### S1 Ch10に追加する2話

**「告白」**（Suke-sanの信頼宣言の直後）
> Obiaがことばを選んでAko-chanに打ち明ける：「ずっと監視していた。データを送り続けていた。宛先は分からない。自分の送信が異変と連動していると思う。」

**「送信停止」**
> ObiaがAko-chanに向かって宣言する：「やめる。今ここで。」  
> プログラムに反する最初の自発的行動 ＝ Obiaが監視者でなく仲間になる瞬間。

### S1→S2の接続

- S1末：数週間の帰還の旅の中で動物の声が静かになっていく。イギリス到着時、ハリネズミが「地面が静かになった」と告げる。
- 「なぜ収まったか」をAko-chanは知らない（Obiaの送信停止が原因だとは分かっていない）。
- S2の信号は「以前の乱れの再来」ではなく、**未来の少年がタイムマシンに仕込んだObiaへの召喚信号**（暗号化）。

### Pod（リクガメ用自走式移動装置）

DaruとGabu（S1から）、Ruka（S2から）の移動装置。「Pod」と呼ぶ。

| 項目 | 内容 |
|------|------|
| 仕組み | 内部でリクガメが歩くと連動してPodが回転移動する（自走式） |
| 個性 | Daruは全力歩行→高速移動、Gabuは一定リズム→安定走行、Rukaはゆっくり止まりながら→控えめな動き |
| ロックモード | 嵐・危険時に切り替わる。中で歩いても動かない（固定） |
| 発光色 | Daru：緑、Gabu：オレンジ、Ruka：黄色 |
| S1サイズ | Daru・Gabu用（コンパクト）|
| 南極モード | S1より大型化、絶熱対応 |
| S2 Ruka用 | Daru/Gabuより小さいミニPod。Ch15（再集結時）に完成。 |

### その他の環境対応ガジェット

| 環境 | 対象 | ガジェット |
|------|------|---------|
| 酷暑（ワディラム砂漠） | Ako-chan | 熱反射クロス（超軽量・熱を跳ね返す素材） |
| 酷暑（ワディラム砂漠） | Dango | 肉球保護ブーツ＋鼻先マスク |
| 南極 | Ako-chan | 超軽量・絶対凍えない未来の防寒服 |
| 南極 | Dango, Suke-san | 保温犬用ブーツ＆ベスト |
| 南極 | Pen-san, General Adelie | 不要（南極の生き物） |

### S1 の南極拠点

S1における南極の拠点は**Obiaが作った船のみ**。建造物・「基地」は存在しない。

- 上陸後も船は沿岸の氷に接岸したまま停泊
- 活動はすべて船周辺の沿岸・氷の上（船から視界の届く範囲）
- 吹雪などの危機時は全員が船内に退避する
- 「基地の中」「基地の外」等の表現はすべて「船の中」「船の外（氷の上）」に統一
- 洞窟の存在はS2 Ch16まで知らない（Ch16で初めて洞窟入口を発見）

## 10. タイムトラベルの詳細（Season 2 Ch18〜20）

### 訪問種（3種）

| 順 | 区分 | 動物 | 時代・場所 | 同行メンバー（Ako-chan・Obia除く） |
|----|------|------|----------|----------------------------------|
| 1 | 意図的 | ニホンオオカミ | 19世紀末〜20世紀初頭・日本の山奥 | Dango・Suke-san |
| 2 | 意図的 | オオウミガラス | 18〜19世紀初頭・北大西洋の孤島 | Pen-san・General Adelie |
| 3 | 事故的 | マイオラニア | 氷河期・オーストラリア周辺 | Ruka・Daru・Gabu（事故的に）、後にAko-chan・Obia |

### 基本ルール

- **Ako-chanとObiaは全ジャンプに必須**（Ako-chanが声を聴き、Obiaが未来へ転送する）
- **メンバーは都度入れ替え**：タイムマシンには乗れる人数の上限がある
- **行き先は跳躍前に告げない**：到着後の情景描写で読者を誘導する
- **Ako-chanの役割**：各絶滅動物の声はAko-chanにしか聴こえない。「声を受け取る」瞬間にObiaがリアルタイムで未来（少年の時代）へ転送する

### Ch19の構造（三度の跳躍）

**Jump 1（ニホンオオカミ）**
- メンバー：Ako-chan・Obia・Dango・Suke-san
- 待機組：Pen-san・Adelie・Daru・Gabu・Ruka
- 声を採取し現代へ帰還→帰還時に軽い異変（時空の揺れ）

**Jump 2（オオウミガラス）**
- メンバー：Ako-chan・Obia・Pen-san・General Adelie
- 待機組：Dango・Suke-san・Daru・Gabu・Ruka
- 声を採取し現代へ帰還→帰還時に異変が拡大（前回より明らかに大きい）

**Jump 3（開始のみ、Ch19末で中断）**
- メンバー（予定）：Ako-chan・Obia・Daru・Gabu・Ruka
- 待機組：Dango・Suke-san・Pen-san・Adelie
- 跳躍を開始したが、移動中に異変が臨界点に達する → Ch20へ続く

### Ch20の構造（2軸交互進行）

**発端**
1. Jump3の移動中、時空の歪みが限界に達しRukaが飲み込まれる
2. DaruとGabuが自分の意思で飛び込んでRukaを追う（躊躇なし）
3. Ako-chanとObiaは現代のタイムマシン基地に取り残される
4. Ako-chan：「Find them!」

**交互進行（5ビート）**

| ビート | 場所 | 内容 |
|--------|------|------|
| 1 | 氷河期オーストラリア | リクガメ3匹が到着・マイオラニアと出会う |
| 2 | 現代（TM基地） | Ako-chan・Obiaが行き先を割り出す（Ako-chanが声の方向を感じ、Obiaがセンサー＆計算で特定）。待機組（Dango・Suke-san・Pen-san・Adelie）が「帰ってこない、何かあったか」と心配するシーン |
| 3 | 氷河期オーストラリア | リクガメとマイオラニアの語らい |
| 4 | 氷河期オーストラリア | Ako-chan・Obiaが到着。5匹と合流 |
| 5 | 氷河期→現代 | Ako-chanがマイオラニアの声を受け取る・Obiaが転送・全員で現代へ帰還 |

**待機組シーンの扱い**：ビート2に1〜2話織り込む。テンポを損なう場合は1話に圧縮してよい。

**核心**：DaruとGabuが「考える前に飛び込む」。普段は慎重なGabuが躊躇なく動く場面で、Rukaへの愛着の深さが示される。マイオラニアの声は計画外だったが、Ako-chanとObiaの連携で採取される。

## 11. タイムループ構造（シリーズ全体）

S3の物語が示す因果ループ：

```
少年がObiaを作る（S3）
  ↓
ObiaがS1の時代に送られる（S3 Ch33）
  ↓
ObiaがAko-chanたちと旅をする（S1〜S2）
  ↓
Ako-chanたちの絆が完成する（S2末）
  ↓
少年がその「完成した絆」を目撃する（S3）
  ↓
少年が「始まりを知りたい」と思う
  ↓
少年がObiaを作る ← ループ
```

S3のテーマ：「完成した絆を目撃した少年」（絆の始まりを目撃するためにObiaが作られる）

## 12. 重要な継続性ルール

- **異変はObiaが到着してから始まった。** どのキャラクターも「ずっと前から」とは言えない。
- **引用台詞は timeline.md の正確な表現を使う。**
- **"For a long time" などの期間表現は要注意。** 必ずObiaの到着タイムラインと照合する。
- **Suke-sanの立ち位置**：常にAko-chanの右・少し後ろ。
- **赤い日記帳**：S1 Ep5で初めて書いた（"I have to go."）。S2 Ep100で「Obia——」と書く。S3への橋渡し。
- **Pod表記**：「カプセル」は使わない。Daru・Gabu・Rukaの移動装置は全て「Pod」。
- **エピソードタイトル**：キャラクター名のみ（*Daru*、*Suke-san* など）や一般的すぎる1語（*Five*、*North* など）は原則禁止。ただし感情的なクライマックスに限り例外として使用可（例：*Obia*、*Stop*）。詳細は `writing-guide.md` Section 4 参照。

## 13. エピソード奥付ルール（frontmatter）

```yaml
---
layout: episode
title: "エピソードタイトル（英語）"
season: 1       # または 2, 3
chapter: 1      # 通しチャプター番号（S1: 1-11, S2: 12-22, S3: 23-33）
episode: 1      # シーズン内エピソード番号
story_day: 1    # 物語内の日付（整数）
level: A1       # または A1-A2, A2-B1
word_count: 0   # 執筆後に実測値に更新
---
```

**word_count の測定コマンド：**
```bash
awk '/^---/{n++; if(n==2){found=1; next}} found' EPISODE_FILE | wc -w
```
- 270語未満 → テクスチャー追加
- 330語超 → トリミング
