## A. 運用細則の継承（G1〜G16以外の確立ルール）

| ルール | 内容 |
|--------|------|
| 人数宣言の集約 | 一行の人数を声に出すのは**列挙イベント回のみ**（例: S1 "Six on Board"型・帰路の "Now they are three"型）。合流のたびに数を宣言しない |
| 数え語の登録 | 本文中の数値・数え語（"four words"・"five metres"・"four boots"等）は台帳登録して逐語管理。数に依存する言い回しは極力避ける |
| "the silver one/thing" | "robot"回避の言い換え。**既定呼称は名前＋it**。異質さ・不気味さを狙う時のみ1話0〜1回。親密ビートでは名前 |
| Pen-sanの先知 | 再会時にAko-chanの名を知っている等の寓話的余白は**説明しない**（「動物と話せる理由は描かない」と同調） |
| ハリネズミ対構図 | 開幕「地面がおかしい」↔ 終幕「地面が静かになった」の対回収はS1の額縁として維持 |
| 南極のリクガメ | 氷上でも船内でもPodから出ない（体温維持）。吹雪時はPodロックモード |
| 夜の犬 | 家の犬（Dango）を夜間に屋外へ単独で立たせない（見張りは窓辺・戸口） |
| 絶滅動物の扱い | 命名しない・"extinct"不使用（"the last pack"等）。実在種名は教材価値で1〜2回まで可（great auk型） |
| タイトル衝突回避 | 類似タイトル（The Empty Place / The Empty Space型）は全季横断で重複チェック |
| story_day | 季開幕でDay1リセット・非減少・厳密管理はしない（"説明しない"方針） |
| 章間リンク | 章頭prev=前章末・前章末next=今章頭・季末→次季頭の連鎖検証 |
