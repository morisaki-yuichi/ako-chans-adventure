## B. 配置・動線

| キャラ・物 | 定位置・所作ルール |
|-----------|-------------------|
| Dango | 常にAko-chanのそば／足元。**夜に屋外単独で立たせない**（見張りは窓辺・戸口）。S3=真ん中 |
| Suke-san | **Ako-chanの右・少し後ろ**（S1 Ch6〜全季不変）。S3で見張りの座の再演（Ep214） |
| Obia | S1=庭の小屋のそば（Ep3の位置・Ep97で回帰）／S2=家族なので夜は屋内可。代名詞it・paw/feet/front legs・"robot"不使用 |
| Dalu／Gabu／Luca | 移動は常にPod（赤／青／黄）。**南極では氷上も船内もPodから出ない**（吹雪時ロックモード）。S2 Ch13のLucaはまだPod無し（ミニPod=Ep106） |
| Luca | S2=縁石の日溜まり・水器のそば・玄関の棚の上／S3=棚の上＋少年の足元へ行く役 |
| S3の定位置 | Dalu=日溜まり／Gabu=木陰／Luca=棚の上／Suke-san=右後ろ／Dango=真ん中 |
| S1南極拠点 | **船のみ**（基地なし・船の中/外で表現）。船から離れない動機を明示（Ep65） |
| 洞窟（S2〜） | 入口=崖の陰の裂け目（Ep127）・長い下降の縦深（最奥に機械）。入口↔最奥の即往復をさせない |
| 温室（S3） | 公園隅・**中立な二人の場所**（少年の所有物にしない） |
| 一行の人数の推移 | S1: 3→(+Dalu Ep21)4→(+Gabu Ep31)5→(+Suke-san)6=**Ep46宣言**→帰路で逓減→3（**Ep95宣言可**）／S2: 4→再集結で7=**Ep121列挙のみ**→Obia去り6／S3: **6＋3Pod固定**（少年は一行に加わらない） |
