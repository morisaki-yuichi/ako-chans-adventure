# Continuity Ledger — 執筆前の参照台帳

形式チェック（語数・文法・frontmatter）では拾えない**内容・論理の矛盾**を未然に防ぐための台帳。
**各話の執筆前にこの3台帳を確認し、執筆後にレビューF（精読）でも照合する。**

3つの台帳：
- **A. 知識状態・命名**（誰が・いつ・何を知るか／固有名がいつ作中で確立するか）
- **B. 配置・動線**（各キャラの定位置と、シーンごとの居場所・移動の妥当性）
- **C. 数・量の登録簿**（本文中の数値・数え語を正確に保つ）

---

## A. 知識状態・命名（Knowledge & Naming）

> 原則：**地の文と台詞は「その時点で確立済みの知識」しか使えない**。名前・事実・能力を、確立前に先取りして呼ばない。

### A-1. 固有名の作中確立点（最重要）

| 名前／呼称 | 確立する回 | それ以前の呼び方 | メモ |
|-----------|-----------|----------------|------|
| **Obia（名前）** | **Ep3** で本人が名乗る：*"I am Obia."* | Ep1–2：「目」「something」等。**名前を出さない** | Ep3以降は地の文で "Obia" 可。※用途・目的は名乗らない（謎のまま） |
| Obiaの正体（未来のロボット／送信者） | S1では明かさない。告白は**Ep83（Ch11）** | — | S1中は「銀色の犬のようなもの」。誰も正体を知らない |
| Daru / Gabu / Suke-san / Pen-san / Adelie | 各登場回（Ep18/31/39/65/67…）で名乗り or Ako-chanが受け取る | 登場前は言及しない | |
| 赤い首輪の正体（Ako-chan作） | **S3まで伏せる** | S1では「Obiaが元から付けている赤い首輪」 | S1で正体を匂わせない |

⚠️ 今回のテストで発生した誤り：Ep3でObiaは名乗らず（書き手は正しく "the silver thing" と表記）、なのにEp5地の文が "Obia's words" と先取り。→ **Ep3でObiaに名乗らせる**ことで根治（下記仕様修正済み）。

### A-2. 能力・事実の確立点（Ch1）

| 事実 | 確立回 | メモ |
|------|--------|------|
| Ako-chanは動物の声が聴ける | Ep1–2（生まれつき・理由は描かない） | Ep2で自然に示す |
| **聴けるのはAko-chanだけ／Dangoは聴こえない** | **Ep2** で明示（"Only Ako-chan can."） | Ep1は伏線（Dangoは見るだけ・吠えない） |
| 異変が存在する | Ep2（ハリネズミ） | **異変はObia到着＝Ep1から**。「ずっと前から」禁止 |
| 異変は南を指す | Ep2–Ep4（ハリネズミ→キツネ/フクロウ） | |
| Obiaの「到着」 | **Ep1（庭の「目」）**＝執筆者向けの継続性事実 | **「目＝Obia」の連結は作中で明言せず、S3 Ep299 まで伏せる**（読者視点の"出現"はEp3）。Ep3は“名乗り” |
| Ako-chanの旅立ちの決意 | Ep5（日記）→ Ep6（出発を告げる）→ Ep7（出発） | |

### A-3. 各話開始時点で Ako-chan が知っていること（Ch1）

- **Ep1**：日常。異変はまだ意識していない（庭で「目」を一瞬見るのみ）。
- **Ep2**：ハリネズミから異変を知る。南方向の気配。
- **Ep3**：銀色のものが現れ、名乗る（Obia）。旅に出ると告げられる。**理由は知らない**。一度断る。
- **Ep4**：異変が広がっている（複数の動物が南を指す）と知る。
- **Ep5**：心が傾く。日記に *I have to go.* と書く。
- **Ep6**：決意。Obiaに「明日発つ」と告げる。
- **Ep7**：出発。

## B. 配置・動線（Blocking）

> 原則：各シーンで**誰がどこにいるか**を意識し、直前からの移動が自然かを確認する。屋内/屋外・昼/夜の整合に注意。

### B-1. キャラ別の定位置・所作

| キャラ | 定位置・癖 | 注意 |
|--------|-----------|------|
| **Dango** | 常に**Ako-chanのそば／足元**。脚に寄りかかる。屋内ではAkoChanと同室。 | ⚠️ **夜に屋外で単独で立たせない**（家の犬）。「見張り」を出すなら室内の窓辺や戸口で。今回テストのEp5「窓の外で立つ」は誤り→室内へ修正済み |
| **Obia** | Ep3では「小屋（shed）のそば」に留まり、近づきも去りもしない | 距離感＝まだ仲間でない |
| **ハリネズミ** | 庭（生垣の根元など）。Ep2に登場、Ep99で再登場（対の構図） | |
| **Suke-san**（S1 Ch6〜） | 常に**Ako-chanの右・少し後ろ** | シリーズ通しの定位置 |
| **Daru/Gabu/Ruka** | 移動は**常にお気に入りのPod**（発光：赤/青/黄） | 直接の接触時のみPodを出る |
| **Pen-san / Adelie**（南極） | 氷の上／船周辺。S1の拠点は**船のみ**（基地なし） | |

### B-2. Ch1のシーン別ステージング

- Ep1：夜明け前の**庭**（屋外）。Ako-chan＋Dango。「目」は庭の隅の影。
- Ep2：同じ朝の**庭**。ハリネズミ登場。
- Ep3：**庭**。Obiaが小屋のそばに現れる。DangoはAko-chanの前／脚に寄る。
- Ep4：**夜**。屋内 or 庭の戸口から。キツネ・フクロウの声（屋外の生き物）。
- Ep5：**夜・室内（Ako-chanの部屋／机）**。ランプ。**Dangoは室内（足元・戸口・窓の内側）**。
- Ep6：**朝・屋内→庭**。荷造り、Obiaへ告げる。
- Ep7：**夜明け前・庭**（屋外）。三人で出発。

## C. 数・量の登録簿（Quantitative）

> 原則：本文中の**数値・数え語**（"three words", "two eyes", "a few days", "all five" 等）は、必ず実物と一致させる。**数に依存した言い回しは、誤りやすいので極力避ける**か、確定値を使う。

| 対象 | 確定値 | メモ |
|------|--------|------|
| 日記の最初の一行 *I have to go.* | **4語** | ⚠️ "three words" は誤り（今回テスト）。数えるなら "four words"、または数えない言い回し（"a line" / "the words"）に |
| Obiaの目 | **2つ**（pale blue） | "two eyes" 可 |
| 一行（party）の人数 | Ch1〜2＝**3**（Ako-chan・Dango・Obia）／Daru合流後（Ep21〜）＝**4**／Gabu合流後（Ep31〜）＝**5**／Suke-san合流後（Ep45〜）＝**6** | Ep7「three of them」可。Ep21 "Now they are four."／Ep31 "Now they are five. Ako-chan, Dango, Obia, Daru, and Gabu."／Ep45 "Six on Board"（六人＝＋Suke-san）。※Pod/Suke-sanは"匹"だが英語本文の通称人数として登録 |
| Gabu出現〜合流の距離・時間（Ep28） | 距離＝**fifteen meters**（タイトル）／待ち時間＝**an hour** | Gabuは岩上15m・全員で1時間待つ |
| Dangoの砂漠ブーツ（Ep26） | **4つ**（"four small boots"）＋鼻先マスク1 | 4本脚に対応 |
| 期間表現 | 厳密管理しない（"説明しない"方針）が、**異変の継続期間を長く言わない**（Obia到着＝Ep1以降） | timeline.md の Day 割付と矛盾させない |

リクガメのサイズ等、シーズン固有の数値は各 episode-list / world.md / s3 対応表を参照。

## 使い方

1. **執筆前**：書く話について A-3（その話で既知のこと）・A-1/A-2（先取り禁止の名前・事実）・B-2（ステージング）・C（数）を確認する。
2. **執筆後**：レビューF（精読・二段台帳照合）でこの台帳と逐文照合する。
3. 新たな確立点・定位置・確定数が生じたら、執筆後にこの台帳へ追記する（timeline.md と二本立てで運用）。
